# Proteccion-del-medio-ambiente

## Introducción

Este GitHub está basado en la protección del medio ambiente, que es, que hacemos que contamina y como podemos pararlo y ayudar con el medio ambiente. La contaminación del medio ambiente es algo que lleva pasando muchos años y aunque es cierto que en los últimos años se están reforzando las medidas de protección, la cantidad de contaminación diaria supera a estas medidas, por lo que hay que seguir luchando dia a dia para que en algún momento sea al revés y podamos decir que tenemos un planeta bien cuidado, lo cual ahora mismo por pena no podemos decir.
